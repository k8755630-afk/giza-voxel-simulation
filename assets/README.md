# Assets

This directory contains project assets including models, textures, data files, and other resources used by the simulation.

## Structure

- `models/` - 3D models and mesh files
- `textures/` - Texture and material files
- `data/` - Simulation data and datasets
- `images/` - Screenshots and promotional images

## Adding Assets

When adding large files, consider using Git LFS (Large File Storage).
