# Configuration

This directory contains configuration files for the Giza Voxel Simulation project.

## Files

- `config.yaml` - Main configuration file
- `default.yaml` - Default configuration template
- `simulation.yaml` - Simulation-specific settings

## Usage

Configuration files use YAML format. The application loads configuration at startup based on environment variables or command-line arguments.

## Environment Variables

- `GIZA_CONFIG_PATH` - Path to custom configuration file
- `GIZA_LOG_LEVEL` - Logging level (DEBUG, INFO, WARNING, ERROR)
