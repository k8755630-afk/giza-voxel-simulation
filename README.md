# Giza Voxel Simulation

A sophisticated voxel-based simulation project focused on archaeological and architectural visualization.

## Project Structure

```
giza-voxel-simulation/
├── src/                    # Source code
├── docs/                   # Documentation
├── assets/                 # Models, textures, and data files
├── tests/                  # Test suite
├── config/                 # Configuration files
└── README.md              # This file
```

## Quick Start

See [Getting Started Guide](docs/GETTING_STARTED.md) for detailed instructions.

```bash
# Clone and setup
git clone https://github.com/k8755630-afk/giza-voxel-simulation.git
cd giza-voxel-simulation
pip install -r requirements.txt

# Run the simulation
python src/main.py
```
